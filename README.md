# ICAE — Indicateur Composite Avancé de l'Économie

Application Streamlit pour le calcul, la prévision et le nowcasting de l'ICAE pour les pays de la CEMAC.

## Modules

1. **Calcul ICAE** — Calcul de l'indicateur composite à partir des fichiers consolidés
2. **Prévisions** — Méthodes de prévision (MM3, MM6, MM12, NS, CS, TL, ARIMA, ETS)
3. **Nowcast** — Modèles de nowcasting (Bridge, U-MIDAS, PC, DFM)
4. **CEMAC Agrégé** — Agrégation pondérée pour la zone CEMAC
5. **Rapports** — Génération de rapports Word

## Utilisation

Uploadez vos fichiers ICAE consolidés (.xlsx) pour démarrer l'analyse.
